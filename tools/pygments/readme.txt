This is a lexer complect for great Python syntax highlighting package named
pygments.

To install pygments see information on http://pygments.org site.

To install malco_lexers:
1) Obtain setuptools [http://peak.telecommunity.com/DevCenter/EasyInstall].
2) Install pygments (if you haven't already done that).
3) Run following command:
$ python setup.py install
